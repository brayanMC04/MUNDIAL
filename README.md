# Polla Mundialista 2026

Aplicación web para gestionar la polla mundialista de la empresa.

## Tecnologías

### Frontend
- React
- Vite
- Bootstrap

### Backend
- Flask
- JWT
- PyMySQL

### Base de datos
- MySQL

## Funcionalidades

- Login por cédula
- Pronósticos de partidos
- Selección de campeón
- Ranking general
- Panel administrativo
